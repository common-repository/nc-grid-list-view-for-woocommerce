== Description ==
This plugin allows you to add list and grid view toggle option to your WooCommerce store.

= Features =
1. Set list or grid as default view.
2. Disable grid list view on categories.
3. List view image position to left or right.
4. Custom width for image and content wrapper for list view.

== Installation Guide ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Follow the setting link from plugin page.
